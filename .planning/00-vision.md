# CafeHub — Vision & Goals

## 🎯 한 줄 가치 제안

**가고 싶은 카페를 찾아서 원하는 시간에 자리를 미리 예약하는 모바일 앱**

---

## 📌 비전 (Vision)

### 문제 정의

현대인들은 카페를 방문할 때 다음 3가지 문제를 겪습니다:

1. **시간 낭비**
   - 카페 도착 후 자리가 없어 돌아감
   - 다른 카페를 찾아야 함
   - 편도 왕복 30분 이상 손실

2. **정보 부족**
   - 카페의 분위기, 시설을 미리 알 수 없음
   - 여러 플랫폼을 찾아봐야 함 (지도, SNS, 리뷰 사이트 산재)
   - 신뢰할 수 있는 최신 정보 부족

3. **계획 불가능**
   - 그룹 미팅이나 데이트 장소 확보 어려움
   - "지금 자리 있나요?" 몇 번이나 문의
   - 예약이 불가능하면 즉흥적으로 결정해야 함

### 우리의 해결책

**CafeHub**는 다음을 제공합니다:

- 🗺️ **지도 기반 검색**: 위치, 분위기, 시설로 카페 필터링
- 📅 **실시간 예약**: 원하는 날짜/시간/인원으로 자리 즉시 예약
- ⭐ **신뢰할 수 있는 정보**: 예약 가능한 카페만 엄선, 사진과 리뷰로 검증
- 💳 **간편 결제**: 예약 시 결제, 현장에서 편하게 입장

### 핵심 가치

| 가치 | 설명 |
|------|------|
| **시간 낭비 제로** | 클릭 3번으로 예약 완료, 도착 전에 자리 확보 |
| **한곳에서 관리** | 카페 정보, 예약, 리뷰 모두 한 앱에 |
| **신뢰성** | 사진과 리뷰로 검증된 카페만 추천 |
| **실시간 업데이트** | 카페의 현재 상태를 실시간으로 파악 |

## 👥 주요 사용자 (Personas)

### 1️⃣ 직장인 (영준, 28세)

**상황**: 점심시간에 팀장님과 카페에서 미팅

**목표**: "2시간 뒤 팀장님과 만나는데, 조용한 카페 자리를 미리 확보하고 싶어"

**행동**:
1. CafeHub 열기
2. "강남역 조용한 카페" 검색
3. 분위기 필터링 (모임용, 분위기 좋음)
4. 평점 높은 카페 선택
5. 날짜/시간 (오후 2시, 4명) 입력
6. 결제 완료 (앱카드 또는 신용카드)
7. 시간에 딱 맞춰 도착 → 즉시 앉기 ✅

**니즈**:
- 빠른 검색과 예약
- 신뢰할 수 있는 정보
- 결제 편의성

---

### 2️⃣ 학생 / 프리랜서 (지은, 24세)

**상황**: 공부 또는 작업을 위해 하루종일 카페에 머물기

**목표**: "집중력 좋은 카페에서 4시간 공부하고 싶어"

**행동**:
1. CafeHub 열기
2. "신림역 스터디 카페" 검색
3. 분위기 필터링 (조용함, WiFi 있음)
4. 사진으로 실제 분위기 확인
5. 리뷰에서 "정말 조용해요" 댓글 확인
6. 이웃한 시간대 예약 (2시~6시)
7. 충전 완료 후 장시간 머물기 ✅

**니즈**:
- 신뢰할 수 있는 분위기 정보
- 충전소, WiFi 등 시설 정보
- 리뷰의 신뢰도

---

### 3️⃣ 커플 (민준&수연, 26세)

**상황**: 주말 데이트로 예쁜 카페에서 2시간

**목표**: "분위기 좋은 카페를 미리 예약하고 편하게 즐기고 싶어"

**행동**:
1. CafeHub 열기
2. "홍대 데이트 카페" 검색
3. 분위기 필터링 (분위기 좋음, 감성적)
4. 사진으로 인테리어 확인
5. 리뷰에서 커플 후기 읽기
6. 토요일 오후 4시 (2명) 예약
7. 시간에 도착 → 대기 없이 즉시 입장 ✅

**니즈**:
- 예쁜 카페 시각적 정보 (사진)
- 분위기 상세 정보
- 신뢰할 수 있는 평가

## 📋 목표 (Goals)

### 1단계: MVP (Minimum Viable Product)

**6주 내에 만족해야 할 기본 기능:**

```
✅ 회원가입/로그인 (Firebase Auth)
✅ 지도에 카페 표시 (Google Maps API)
✅ 카페 검색 & 필터링
✅ 카페 상세 정보 보기
✅ 자리 예약 (날짜/시간/인원)
✅ 결제 (Stripe)
✅ 예약 확인/취소
✅ 카페 리뷰 & 평가
```

### 2단계: 추가 기능 (Should Have)

```
- 찜한 카페 저장
- 카페 방문 기록
- 친구와 예약 공유
- 카페별 통계 (방문객, 만족도)
```

### 3단계: 미래 (Won't Have - 이번 프로젝트에서는 안 함)

```
- 결제 시 할인/쿠폰
- 카페 사업자 관리 페이지
- 음성 주문
- AI 추천 엔진
```
