## 📌 세부 분석

### 1. 사용자 관리 영역 (5일)

#### 1.1 회원가입 화면 (2d)

**산출물**: 
- `lib/presentation/screens/signup_screen.dart`
- Firebase Auth 연동된 회원가입 로직

**상세 작업**:
```
Day 1:
  - UI 디자인 (이메일, 비밀번호, 이름 입력)
  - Flutter TextFormField 구현
  - 입력 검증 로직 (이메일 형식, 비밀번호 길이)

Day 2:
  - Firebase Authentication 연동
  - 회원가입 API 호출
  - 에러 핸들링 (이미 있는 계정 등)
  - 가입 완료 후 로그인 화면 전환
```

**체크리스트**:
- [x] UI 렌더링 완료
- [x] 입력 필드 검증
- [x] Firebase 연동
- [x] 에러 메시지 표시
- [x] 성공 시 자동 로그인

**상세 작업**:
```
Day 1:
  - Firestore Collection "Cafes" 설계
  - Document 필드 정의 (이름, 주소, 위도, 경도, 분위기, 시설, 사진 등)
  
Day 2:
  - 초기 데이터 50개 입력 (수동 또는 스크립트)

Day 3:
  - Cafe 모델 클래스 구현

  - 데이터 조회 API 구현 (Repository)
```
