### 중간발표 데모 시나리오

```
시연 순서:

1️⃣ 회원가입 (20초)
   - 이메일 입력
   - 회원가입 완료
   - 자동 로그인

2️⃣ 지도 보기 (30초)
   - 강남역 근처 카페 3~5개 표시
   - 마커 클릭 시 카페명 표시

3️⃣ 카페 상세보기 (40초)
   - 한 카페 선택
   - 사진, 평점, 정보 표시
   - "예약" 버튼 까지는 아직 안 됨 (다음 주)

3️⃣ 검색해보기 (20초)
   - "강남역 조용한 카페" 검색
   - 결과 표시

총 시간: 약 2분 (충분히 짧음)
```

### 체크리스트

```
기능
□ 회원가입 (이메일)
□ 로그인 (이메일, Google)
□ 프로필 조회
□ 프로필 수정 (선택)
□ 지도 표시
□ 카페 마커 (50개)
□ 카페 상세 정보
□ 카페 사진 갤러리
□ 평점 & 리뷰 조회

품질
□ UI 오류 없음
□ 크래시 없음 (데모 중)
□ 로딩 시간 < 3초
□ 반응형 UI (portrait/landscape)

발표 준비
□ 데모 시나리오 3회 리허설
□ 백업 폰 준비
□ 인터넷 연결 테스트
□ 발표자료 (슬라이드 또는 포스터)
```

### 산출물

  └─ 기본 UI 완성 ✅
